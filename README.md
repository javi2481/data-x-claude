# DataFluxIT - Explorex Platform

Plataforma de exploración de datos asistida por IA.

## Pruebas
Sigue este orden para las pruebas del sistema:
1. `Maquina_expendedora_1.csv` (Simple, datos limpios, validación inicial)
2. `Bitcoin_Price_Dataset_2014_2023.csv` (Series temporales)
3. `student_academic_performance.csv` (Clasificación/Categorización)
4. `genz_mental_wellness.csv` (Análisis de sentimientos/encuestas)
5. `cs_students.csv` (Datos mixtos)
6. `UCI_Credit_Card.csv` (Dataset grande, estrés de performance)

## Setup
Consulta los archivos `.env.example` en `core/backend` y `apps/explorex/backend` para configurar las variables de entorno necesarias.
